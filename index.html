Note: this calculator is a simple implementation and does not support advanced operations or any input longer then 2 elements (For ex.: 2+2+4 here it calculates the last 2 elements and it will calculate only the 2+4).

･ To use it just run:

git clone https://github.com/Muaves/lszszv


And then go into your folder:

cd lszszv


And finally:

xdg-open calc.htm



You can get more information or download it manually in the GitHub repository.



Note: The calculator under 260px x 483px screen width/height is useless.
