# Lszszv - Legszarabb számológéb a világon

Számológép - Calculator

Built with JS and HTML

To use it just run:

```git clone https://github.com/Muaves/lszszv ```

And then go into your folder:
```cd lszszv```

and finally: 
```xdg-open calc.htm```

OR if you want to open it in an exact browser
```firefox calc.htm```
```(your browsers name here) calc.htm```
